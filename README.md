# Bluestock Mutual Fund Capstone

This project contains the data, notebooks, and scripts for the Bluestock Mutual Fund Capstone project.

## Directory Structure

- `data/`: Contains raw, processed, and database files.
- `notebooks/`: Jupyter notebooks for data ingestion, cleaning, EDA, and analytics.
- `scripts/`: Python scripts for ETL, live NAV fetching, metrics computation, and recommendation.
- `sql/`: SQL schema and query files.
- `dashboard/`: PowerBI dashboard files.
- `reports/`: Final reports and presentations.
